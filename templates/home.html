Welcome to bbassyst - beachbody Asset Reporting system!

This application lists assets in beachbody

use the following posts to get details of Assets

  • /assets : To list all assets in the system.
  • /assets/ : to list details of a particular asset by id.
  • /assets/: to list details of a particular asset by name.
  • /env: to list assets acress all environments by environment.
  • /env/: to list assets for a particular environment.
  • /app : to list assets for a all applications for all environments.
  • /app/:to list assets for a particular application across all environments.
